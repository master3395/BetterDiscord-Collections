# Link-Profile-Picture
*(plugin header: `1.3.3`)*

## About

Lets you click users' avatars on their profile page to view a bigger version in your browser.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Install (from this collection)
Copy `linkProfilePicture.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
