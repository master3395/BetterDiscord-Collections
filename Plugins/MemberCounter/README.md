# MemberCounter
*(plugin header: `2.34`)*

## About

Displays the Member Count of a Server at the top of the Member List, can be configured to show Total Members, Online Members, Offline Members, and a DM Counter.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Install (from this collection)
Copy `MemberCounter.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
