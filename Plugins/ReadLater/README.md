# Read Later
*(plugin header: `1.2.21`)*

## About

Local-only queue of message links (Read later). Not channel tabs - manage from plugin settings or open jump links.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Features (overview)

- **Local queue** - saved items live in BetterDiscord data on your machine (`BdApi.Data`), not on Discord’s servers.
- **Jump links** - store messages via context menu / actions and open them later from the list.
- **Guild bar / shortcuts** - quick access from the UI and keyboard where the plugin adds them.
- **Pagination** - optional settings to keep large lists manageable.
