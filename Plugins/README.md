# Plugins

Each subfolder is one plugin (`*.plugin.js`), a **README.md** (about, dependencies, install URL, upstream credits), and a **CHANGELOG.md** (dated **DD.MM.YYYY** entries for edits in this collection). Layout is similar to [Farcrada/DiscordPlugins](https://github.com/Farcrada/DiscordPlugins/tree/master/Double-click-to-edit).
