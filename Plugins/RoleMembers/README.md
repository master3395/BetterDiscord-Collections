# RoleMembers
*(plugin header: `0.2.1`)*

## About

Allows you to see the members of each role on a server.

## Dependencies

- **[ZeresPluginLibrary](../0PluginLibrary/)** - install `0PluginLibrary.plugin.js`, enable it, then enable this plugin.

## Install (from this collection)
Copy `RoleMembers.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
