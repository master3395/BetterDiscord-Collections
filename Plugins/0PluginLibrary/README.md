# ZeresPluginLibrary
*(plugin header: `2.0.23`)*

## About

Gives other plugins utility functions.

## What this is

**ZeresPluginLibrary** (`0PluginLibrary.plugin.js`) provides utilities for plugins that target its API (for example several Zerebos plugins).

Enable this plugin when another plugin’s documentation or startup notice says ZeresPluginLibrary is required.
