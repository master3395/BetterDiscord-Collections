# DuplicateSendGuard
*(plugin header: `1.0.3`)*

## About

Warns before sending a message that matches your own recent message in the same channel.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Features (overview)

- Compares your outgoing message to your **own recent messages** in the same channel.
- **Configurable lookback** and optional whitespace normalization (see plugin settings).
- Shows a **warning** before send so you can cancel or send anyway.
