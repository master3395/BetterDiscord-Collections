# HideDisabledEmojis
*(plugin header: `0.1.0`)*

## About

Hides disabled emojis from the emoji picker.

## Dependencies

- **[ZeresPluginLibrary](../0PluginLibrary/)** - install `0PluginLibrary.plugin.js`, enable it, then enable this plugin.

## Install (from this collection)
Copy `HideDisabledEmojis.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
**Raw (main branch):**
