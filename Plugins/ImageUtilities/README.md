# ImageUtilities
*(plugin header: `5.6.6`)*

## About

Adds several Utilities for Images/Videos (Gallery, Download, Reverse Search, Zoom, Copy, etc.)

## Dependencies

- **[BDFDB](../0BDFDB/)** - install `0BDFDB.plugin.js` from this collection (or upstream), enable it, then enable this plugin.

## Install (from this collection)
Copy `ImageUtilities.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
**Raw (main branch):**
```text
https://raw.githubusercontent.com/master3395/BetterDiscord-Collections/main/Plugins/ImageUtilities/ImageUtilities.plugin.js
```
**This folder in the repo:**
