# MemberCount
*(plugin header: `3.0.17`)*

## About

Displays a server's member-count at the top of the member-list, can be styled with the `#MemberCount` selector.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Install (from this collection)
Copy `MemberCount.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
