# ShowHiddenChannels
*(plugin header: `0.6.8`)*

## About

A plugin which displays all hidden Channels and allows users to view information about them, this won't allow you to read them (impossible).

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Notes

- Shows **metadata** about channels you cannot access; it does **not** let you read messages in those channels (that is not possible client-side).
