# BDFDB
*(plugin header: `4.4.9`)*

## About

Required Library for DevilBro's Plugins

## What this is

**BDFDB** (BetterDiscord) is DevilBro’s shared library. Many plugins in this collection depend on it for UI helpers, settings panels, and Discord module access.

Install and enable **BDFDB** before enabling plugins that show a “library missing” message.

## Install (from this collection)
Copy `0BDFDB.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
**Raw (main branch):**
```text
https://raw.githubusercontent.com/master3395/BetterDiscord-Collections/main/Plugins/0BDFDB/0BDFDB.plugin.js
```
