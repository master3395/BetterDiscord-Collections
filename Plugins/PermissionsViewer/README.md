# PermissionsViewer
*(plugin header: `1.0.0`)*

## About

Allows you to view all the permissions for users, servers, and channels!

## Dependencies

- **[ZeresPluginLibrary](../0PluginLibrary/)** - install `0PluginLibrary.plugin.js`, enable it, then enable this plugin.

## Install (from this collection)
Copy `PermissionsViewer.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
**Raw (main branch):**
