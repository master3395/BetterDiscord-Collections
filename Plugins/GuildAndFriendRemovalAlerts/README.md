# GuildAndFriendRemovalAlerts
*(plugin header: `3.2.1`)*

## About

Displays alerts when you are kicked/banned from a server, a server is deleted, and when a friend removes you.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Install (from this collection)
Copy `GuildAndFriendRemovalAlerts.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
