# Double Click To Edit
*(plugin header: `9.4.10`)*

## About

Double click a message you wrote to quickly edit it.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Notes

- Inspired by Jiiks’ original “double click to edit” idea; Farcrada’s version is maintained separately.
- Check the plugin settings for optional behaviour (for example double-click to reply, if offered in your build).

## Install (from this collection)
Copy `DoubleClickToEdit.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
