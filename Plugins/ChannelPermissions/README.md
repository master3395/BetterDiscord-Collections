# ChannelPermissions
*(plugin header: `4.2.4`)*

## About

Hover over channels to view their required permissions. Massive thanks to Strencher for the help.

## Dependencies

No separate library plugin is required for typical use (BetterDiscord only), unless a plugin update or changelog says otherwise.

## Install (from this collection)
Copy `ChannelPermissions.plugin.js` from this folder into your BetterDiscord plugins directory, or use the raw URL below.
